Ещё одна полезная утилита, предназначенная для пакета Microsoft Word, — Batch Replacer for MS Word — служит для пакетной замены информации в документах Microsoft Word. По сути, программа является плагином для пакета Useful File Utilities (UFO) и работает лишь при установленном пакете. Используя UFO, вначале необходимо выделить все файлы, в которые требуется внести изменения. Затем выбрать в меню файлового менеджера пункт Plug-ins и запустить Batch Replacer for MS Word.
